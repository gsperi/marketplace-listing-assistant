## Project Status

**Current Phase:** Project Definition

**Project Maturity:** Pre-MVP

---

# Vision

Marketplace Listing Assistant is a software platform designed to assist sellers in creating high-quality marketplace listings for platforms such as Vinted, eBay and future marketplaces.

The project does not aim to replace the seller, but to reduce the time required to create effective listings while improving their visibility, quality and consistency.

Starting from a small amount of information, such as product photos or manually entered data, the system assists the user by generating optimized listing content and providing useful suggestions throughout the creation process.

The long-term vision is to become an intelligent assistant capable of simplifying the publishing workflow while remaining independent from any specific marketplace.

---

# Problem Statement

Creating a marketplace listing is often a repetitive and time-consuming activity.

Users are expected to:

* write an effective title;
* produce a clear description;
* choose meaningful keywords;
* estimate a competitive selling price;
* adapt the listing to different marketplaces.

Most sellers are not familiar with the search algorithms used by marketplace platforms and therefore create listings that are incomplete or poorly optimized.

The result is lower visibility, longer selling times and inconsistent listing quality.

---

# Proposed Solution

Marketplace Listing Assistant assists the user throughout the listing creation process.

Starting from product images or manually entered information, the system should be able to:

* identify or classify the product;
* enrich the available information;
* suggest a competitive price range;
* generate optimized titles and descriptions;
* suggest relevant keywords;
* evaluate listing quality through a scoring system;
* export marketplace-ready content.

The user always remains in control of the final listing.

The software provides suggestions, not automatic decisions.

---

# Objectives

The project aims to:

* reduce listing creation time;
* improve listing quality;
* help inexperienced sellers create effective listings;
* provide measurable improvements to the listing creation process;
* remain extensible to additional marketplaces and product categories.

---

# Scope

The primary objective of the project is listing optimization.

The software focuses on helping users create better listings rather than managing the entire selling process.

---

# Out of Scope

The project is not intended to become:

* an ERP system;
* an inventory management platform;
* a complete e-commerce solution;
* an automatic repricing engine;
* a marketplace replacement;
* a fully automated publishing platform.

These capabilities may be evaluated in the future only if they support the core vision of the project.

---

# Minimum Viable Product (MVP)

The initial version should provide the following capabilities:

* upload product images;
* allow manual data entry;
* generate optimized titles;
* generate listing descriptions;
* suggest keywords;
* evaluate listing quality through a score;
* export the generated listing.

---

# Future Evolution

Future versions may introduce:

* marketplace API integrations;
* AI-based product recognition;
* OCR capabilities;
* historical price analysis;
* multi-user support;
* listing performance analytics;
* publication automation where technically appropriate.

These features are intentionally excluded from the initial MVP.

---

# Success Metrics

The project will be considered successful if it can demonstrate one or more of the following:

* reduce listing creation time by at least 50%;
* improve listing completeness;
* generate titles judged better than manually written ones;
* increase listing visibility (to be validated);
* improve conversion rate (to be validated).

These metrics represent project goals and will require validation using real users and real marketplace data.

---

# Guiding Principle

The project focuses on solving a real user problem before introducing technical complexity.

Every new feature should contribute to improving the quality or efficiency of marketplace listing creation.

Technology is a means to achieve this goal, not the goal itself.
